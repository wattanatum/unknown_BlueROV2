# Unknown BlueROV2

Unknown BlueROV2 is a ROS 2 Jazzy underwater robot simulation project using Gazebo Harmonic, BlueROV2, SLAM Toolbox, Nav2, EKF sensor fusion, DVL, IMU, depth control, and autonomous navigation.

## Features

- ROS 2 Jazzy
- Gazebo Harmonic underwater simulation
- BlueROV2 robot model
- DVL, IMU, and depth sensor fusion with EKF
- SLAM Toolbox mapping
- Nav2 autonomous navigation
- Depth control
- Thruster mixer control
